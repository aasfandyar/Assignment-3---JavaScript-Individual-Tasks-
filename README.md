# Assignment-3---JavaScript-Individual-Tasks-

### How to Use
1. Click the main button to increment the counter
2. Use control buttons for additional functionality
3. View statistics in real-time
4. Check click history for past actions
5. Use keyboard shortcuts:
   - Space/Enter: Click
   - Ctrl+R: Reset
   - Shift+=: Add 10 clicks
   - -: Subtract 5 clicks
